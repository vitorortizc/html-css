Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica, ou seja, não possuí significado, mas sim forma).

Nesta frase tenho um termo em destaque usando a tag <strong>.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>(não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em>(não semântica).

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase para indicar que ele foi adicionado depois.

Texto Sublinhado

Existe também um texto sublinhado com a tag <u> (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3 utilizamos a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H20 utilizamos a tag <sub>.

Semântica: O significado dos vocábulos, por oposição à sua forma (sendo o CSS responsável pela forma).
